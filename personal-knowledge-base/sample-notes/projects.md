# Current Projects

## MCP Tutorial Repository

Building a comprehensive tutorial repository with three example MCP servers:

1. **Docker Dev Assistant** - Simple project showing basic MCP concepts
2. **Personal Knowledge Base** - Intermediate project with vector search
3. **Smart Day Planner** - Advanced project with API integrations

The goal is to teach MCP through practical, real-world examples that developers can learn from and extend.

### Technologies
- Python 3.10+
- MCP SDK
- ChromaDB for vector search
- Sentence transformers for local embeddings

### Status
In active development. Docker Dev Assistant is complete, working on Knowledge Base now.
